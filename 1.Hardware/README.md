# 硬件设计文件

本目录包含TARS机器人的硬件设计相关文件。

## 📁 目录结构

```
1.Hardware/
├── PCB/                 # PCB设计文件
├── Schematic/           # 原理图文件
├── Components/          # 元器件清单
├── Assembly/            # 装配图纸
└── BOM/                 # 物料清单
```

## 🔧 主要硬件组件

- **主控板**: ESP32-S3开发板
- **显示屏**: OLED显示屏（128x64）
- **舵机**: SG90微型舵机
- **电源**: 锂电池 + 升压模块
- **传感器**: 麦克风模块、按键开关
- **连接器**: 各类接插件和线材
